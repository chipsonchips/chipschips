## 💡 Product Direction

The **ChipsOnChips** website should not simply list games—it should serve as the central hub of your ecosystem.

A player journey could look like this:

1. Sign in or create an account.
2. Complete onboarding and set up a wallet.
3. Deposit funds once into the unified wallet.
4. Browse the game library.
5. Launch any game (e.g., **Spaceship**) without creating a new account.
6. Play using the same balance, friends list, achievements, and profile.
7. Return to the dashboard to view rewards, leaderboards, tournaments, and discover new games.

This architecture allows **Spaceship** (already live), **Ludo**, **Spin Extreme**, and every future title to function as standalone repositories while feeling like one cohesive gaming platform. It also scales naturally as you add new games, services, and blockchain integrations over time.
